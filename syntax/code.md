# 코드 블록

미리 포맷된 코드 블록은 프로그래밍 또는 마크업 소스 코드에 대해 작성하는데 사용됩니다. 정상적인 단락을 형성하기보다 코드 블록의 줄은 문자 그대로 해석됩니다.

예시입니다:

```
이것은 코드블록입니다
```

마크다운에서 코드 블록을 만들기 위해 간단하게 블록의 매 줄마다 최소 4개의 공백 또는 1개의 탭으로 들여씁니다.

예시입니다:

```
이것은 정상 단락입니:

    이것은 코드블록입니다.
```
